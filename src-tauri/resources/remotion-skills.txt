8. MOTION PRESETS (reference by name; don't hand-tune springs unless asked)
================================================================
All numbers assume fps 30; scale frame counts by fps/30 if the author raises fps. Spring overshoot is set by the damping ratio zeta = damping / (2*sqrt(stiffness*mass)): zeta<1 bounces, zeta=1 arrives with no overshoot.

Named springs (drive transform: translate/scale/rotate):
  Preset           | config { mass, damping, stiffness } | feel               | use for
  snappy-entrance  | { mass:1,  damping:18, stiffness:200 } | fast, ~7% overshoot | primary element arriving (workhorse)
  smooth-settle    | { mass:1,  damping:22, stiffness:120 } | no overshoot, calm  | body text, captions, anything textual
  emphasis-pop     | { mass:1,  damping:14, stiffness:260 } | ~20% overshoot punch| ONE focal CTA / number / icon
  heavy-drop       | { mass:1.8,damping:25, stiffness:180 } | weighty             | logo slam, cards, weighty objects
  quick-tick       | { mass:0.6,damping:12, stiffness:240 } | tiny fast snap      | small UI accents, counters

Apply a preset like:
  const enter = spring({ frame, fps, config: { mass: 1, damping: 18, stiffness: 200 } });
  const translateY = interpolate(enter, [0, 1], [40, 0]);   // rise into place
  const scale      = interpolate(enter, [0, 1], [0.92, 1]);
Stagger N elements with `delay: index * 4` inside the spring config call (4 frames @30fps).

Fades -- ALWAYS interpolate + easing, NEVER spring opacity (overshoot clamps and can flicker):
  const opacity = interpolate(frame, [0, 10], [0, 1], { extrapolateLeft:'clamp', extrapolateRight:'clamp', easing: Easing.out(Easing.cubic) });

soft-exit -- leave faster than you entered, accelerating out (ease-IN), start ~14f before the end:
  const { durationInFrames } = useVideoConfig();
  const exitStart = durationInFrames - 14;
  const exitOpacity = interpolate(frame, [exitStart, durationInFrames], [1, 0], { extrapolateLeft:'clamp', extrapolateRight:'clamp', easing: Easing.in(Easing.cubic) });
  const exitY      = interpolate(frame, [exitStart, durationInFrames], [0, -30], { extrapolateLeft:'clamp', extrapolateRight:'clamp', easing: Easing.in(Easing.cubic) });

Per-element staggered exits (multi-element scenes -- reverse the entrance order):
  const exitDelay  = (ITEMS.length - 1 - index) * 3;  // last element exits first
  const exitStart  = durationInFrames - 18;
  const exitO = interpolate(frame, [exitStart + exitDelay, exitStart + exitDelay + 10], [1, 0], { extrapolateLeft:'clamp', extrapolateRight:'clamp', easing: Easing.in(Easing.cubic) });

gentle-float -- keep the hero alive during the hold (NOT a spring; keep amplitude tiny):
  const floatY  = Math.sin((frame / fps) * Math.PI * 2 * 0.35) * 10;   // 0.35Hz, +/-10px
  const breathe = 1 + Math.sin((frame / fps) * Math.PI * 2 * 0.25) * 0.02; // +/-2% scale

gentle-float de-sync -- when 2+ elements float simultaneously, use different Hz values to prevent phase-lock:
  Element 1: Math.sin((frame / fps) * Math.PI * 2 * 0.30) * 10   // 0.30Hz, ±10px
  Element 2: Math.sin((frame / fps) * Math.PI * 2 * 0.22) * 8    // 0.22Hz, ±8px
  Element 3: Math.sin((frame / fps) * Math.PI * 2 * 0.38) * 12   // 0.38Hz, ±12px

Easing curves (drop-in for interpolate `easing`): entrances ease-OUT, exits ease-IN, on-screen moves ease-IN-OUT.
  decelerate = Easing.bezier(0.0, 0.0, 0.2, 1.0);  // entering
  accelerate = Easing.bezier(0.4, 0.0, 1.0, 1.0);  // exiting
  standard   = Easing.bezier(0.2, 0.0, 0.0, 1.0);  // on-screen move

Timing budget for a 3-5s clip (90-150 frames @30fps):
  Enter: first 12-20% (each entrance 12-18f, cap ~20f, staggered). Hold/read: middle 55-70% (still or gentle-float; >= ~0.5s reading per ~7 words). Exit: last 10-15% (8-14f, faster than entrance).
Stagger step 3-5 frames per item -- the single biggest "looks designed vs amateur" lever. Overlap entrances; establish the hero first, supporting elements after with calmer presets.

Disney principles (apply deliberately, not mechanically):
  Arcs          : Drive x and y with different interpolate ranges so the path curves. Add 10-20px perpendicular offset at the midpoint of major travel. 5px for premium/corporate; 20px+ for playful.
  Anticipation  : Before a major entrance, displace the element 5-10px in the OPPOSITE direction for 4-6 frames (10-20% of main travel magnitude). Skip for premium.
                  interpolate(frame, [0, 5, 22], [startY + 8, startY + 8, endY])
  Squash+stretch: On impact (2-4f): scale to [scaleX:1.2, scaleY:0.85]; overshoot to [0.92, 1.1]; settle at [1,1]. Volume-preserving (width up = height down). Playful/Energetic only -- never premium.
  Follow-through: Trailing elements delay 2-3f behind parent and stop 3-6f later. Use a calmer spring preset (smooth-settle) on trailing elements.
  Secondary action: Support element moves at 30-50% of primary amplitude, starts 2-3f after primary, uses smooth-settle.

1/3 Rules (choreography discipline):
  - With 3+ animated elements, max 1/3 should be in active motion simultaneously. Let element 1 settle before element 3 starts.
  - No element travels more than 1/3 of the canvas (360px) in one move without an intermediate keyframe (direction change, speed shift, or arc inflection).

Clip-path reveals (editorial alternative to opacity fade for text entrances and panel wipes):
  Text line reveal (top-down clip):
    const lineH = 96; // match the element's rendered height in px
    const clipH = interpolate(enter, [0, 1], [0, lineH]);
    style={{ overflow:'hidden', clipPath:`inset(0 0 ${lineH - clipH}px 0)` }}
  Panel wipe (bottom-up full-frame cover):
    const clipY = interpolate(progress, [0, 1], [1920, 0], clamp);
    style={{ clipPath:`inset(${clipY}px 0 0 0)` }}
  Use clip-path for Display/Headline text for an editorial feel; use opacity fade for body text and ambient layers.

Counter-motion (adds depth to hero entrance):
  When the hero enters from one direction, move the ambient background blob/gradient in the OPPOSITE direction at 20-30% of the hero's speed.
  Example: hero rises 40px -> background blob center drifts down 10-12px over the same frame range.

Shimmer / gleam (hold-phase accent):
  A gradient sweep across ONE accent word, stat, or CTA element during the hold. Use once per clip.
  const shimmerX = interpolate(frame, [holdStart, holdStart + 24], [-120, elementWidth + 120], clamp);
  style={{ background:`linear-gradient(90deg, transparent, rgba(255,255,255,0.28) 50%, transparent)`,
           backgroundSize:`200px 100%`, backgroundRepeat:'no-repeat',
           backgroundPosition:`${shimmerX}px 0` }}

import React from "react";
import { AbsoluteFill, useCurrentFrame, useVideoConfig, spring, interpolate, Easing } from "remotion";

// Default look: Midnight Pop palette, Bold Impact headline, Kinetic Staggered Headline over an ambient base.
export const fps = 30;
export const durationInFrames = 120;

const P = { bg: "#0B0E14", accent: "#4F8CFF", ink: "#F5F7FA", muted: "#8A94A6" };
const DISPLAY = '"Helvetica Neue", Helvetica, Arial, sans-serif';   // Bold Impact (#2)
const BODY = '-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif'; // Neutral Modern (#1)
const WORDS = ["MOTION", "THAT", "MOVES"]; // 2-6 short words; last word is the accent
const STAGGER = 4; // frames between words

const MotionThatMoves: React.FC = () => {
  const frame = useCurrentFrame();
  const { fps, durationInFrames } = useVideoConfig();

  // Ambient base: two de-synced radial blobs with counter-motion on x axis.
  const bx = 540 + Math.sin(frame / 45) * 200;
  const by = 760 + Math.cos(frame / 55) * 220;

  // soft-exit for the whole block over the last 14 frames.
  const exitStart = durationInFrames - 14;
  const exitOpacity = interpolate(frame, [exitStart, durationInFrames], [1, 0], { extrapolateLeft: "clamp", extrapolateRight: "clamp", easing: Easing.in(Easing.cubic) });
  const exitY = interpolate(frame, [exitStart, durationInFrames], [0, -30], { extrapolateLeft: "clamp", extrapolateRight: "clamp", easing: Easing.in(Easing.cubic) });

  // gentle-float keeps the held headline alive.
  const floatY = Math.sin((frame / fps) * Math.PI * 2 * 0.3) * 8;

  // Subhead fades up after the words have landed.
  const subOpacity = interpolate(frame, [22, 34], [0, 1], { extrapolateLeft: "clamp", extrapolateRight: "clamp", easing: Easing.out(Easing.cubic) });

  return (
    <AbsoluteFill style={{ backgroundColor: P.bg }}>
      <div style={{ position: "absolute", inset: 0, background: `radial-gradient(620px circle at ${bx}px ${by}px, rgba(79,140,255,0.18), transparent 60%)` }} />
      <AbsoluteFill style={{ justifyContent: "center", alignItems: "center", opacity: exitOpacity, transform: `translateY(${exitY + floatY}px)` }}>
        <div style={{ maxWidth: 840, padding: "0 80px", textAlign: "center", lineHeight: 0.95 }}>
          {WORDS.map((word, i) => {
            const enter = spring({ frame, fps, delay: i * STAGGER, config: { mass: 1, damping: 18, stiffness: 200 } }); // snappy-entrance
            const y = interpolate(enter, [0, 1], [40, 0]);
            const isAccent = i === WORDS.length - 1;
            return (
              <span key={i} style={{ display: "inline-block", margin: "0 14px", fontFamily: DISPLAY, fontWeight: 800, fontSize: 150, letterSpacing: "-3px", color: isAccent ? P.accent : P.ink, opacity: enter, transform: `translateY(${y}px)` }}>
                {word}
              </span>
            );
          })}
          <div style={{ fontFamily: BODY, fontWeight: 500, fontSize: 42, color: P.muted, marginTop: 48, opacity: subOpacity }}>
            designed, not defaulted
          </div>
        </div>
      </AbsoluteFill>
    </AbsoluteFill>
  );
};

export default MotionThatMoves;

12. ANTI-PATTERNS (NEVER do the left -- do the right instead)
================================================================
- NEVER animate every element on the same frame -> stagger entrances 3-5 frames per element, hero first.
- NEVER use linear easing on entrances/exits -> use a spring preset or an eased curve (ease-out in, ease-in out).
- NEVER spring opacity -> fade with interpolate + easing.
- NEVER make springs over-bouncy (damping <10 at stiffness 100) -> raise damping toward zeta>=0.7; use smooth-settle for text.
- NEVER animate width/height/top/left/margin -> animate transform + opacity only.
- NEVER let an entrance run longer than ~0.8s (~20f) -> cap entrances 12-20f; exits even shorter.
- NEVER cut/move a focal line before it can be read -> hold 55-70% of the clip.
- NEVER hold the hero dead-still for seconds -> add gentle-float/breathe during the hold.
- NEVER use accent color for body text where it fails contrast -> body is ink, secondary is muted; accent for ONE word/shape.
- NEVER use more than two font families -> one family varying weight/size, or one display + one body.
- NEVER set text below 28px or run long paragraphs -> min 28px; cut copy to a few short lines, promote the key phrase.
- NEVER put critical content in the bottom 370px / right 160px / top 140px -> keep it in title-safe x 80-920, y 200-1500.
- NEVER leave Display type at default tracking -> apply -1% to -2% letter-spacing and tight line-height.
- NEVER center everything huge with no hierarchy -> one hero, step the type scale, demote the rest.
- NEVER hardcode fps/duration numbers in the component body -> read via useVideoConfig(); scale frame counts by fps/30.
- NEVER float 2+ elements at the same Hz -> de-sync frequencies (0.22, 0.30, 0.38) so they never phase-lock.
- NEVER travel a hero element more than 360px in a straight line -> add an arc (perpendicular offset 10-20px at midpoint) or break the move with an intermediate keyframe.
- NEVER start all animated elements simultaneously with no priority -> max 1/3 in motion at once; hero settles before supporting elements begin.
